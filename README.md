# Claude AI

Constraints for Claude AI to customize Hubleto projects with custom apps.

The [](.claude/CLAUDE.md) file describes constraints for the Claude AI agent specifying how to create custom Hubleto apps in your project.

It is an experimental repository.

## How to use

* create the Hubleto project with `composer create-project hubleto/erp-project`
* download [](.claude/CLAUDE.md) into your project folder
* add description of the app you want to generate into downloaded CLAUDE.md file
* use Claude AI agent to generate your custom app
